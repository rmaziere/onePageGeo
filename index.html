Service de géocodage et géocodage inverse

Ce site utilise fièrement OpenStreetMap, Leaflet et la base adresse nationale.

Géocodage :
Adresse → Géométrie

  1. Vous renseignez une adresse et vous validez,
  2. L'adresse est trouvée : elle est affichée sur la carte.
    La recherche n'est pas probante : une liste est proposée.

Géo-localisation

En se bassant sur :

  • votre localisation,
  • votre adresse IP,

il est possible d'afficher votre position présumée sur la carte.

Géocodage inverse :
Géométrie → Adresse

  1. Vous renseignez une géométrie au format (latitude,longitude),
  2. Elle s'affiche sur la carte,
  3. L'adresse correspondante est retournée.

A propos

Ce mini-site utilise le projet base adresse nationale pour réaliser le géocodage des adresses et le géocodage inverse.
L'affichage est fait via Leaflet avec la cartographie OpenStreetMap.
Bootstrap est aussi dans la place !
